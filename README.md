# AI Engineering Roadmap

Learning workspace for large language model application engineering, AI systems, and MLOps.

## Project Structure

- `notes/`: Daily learning notes.
- `exercises/`: Small practice scripts and environment checks.
- `projects/`: Hands-on projects built during the roadmap.
- `weekly-reviews/`: Weekly summaries, reflections, and next-step plans.

## Quick Start

Run the first environment check:

```bash
python3 exercises/week01/environment_check.py
```
